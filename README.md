# DigitalTutor 🎓

**Система управления учебными проектами для преподавателей вузов**

## 🎯 Зачем это нужно?

DigitalTutor решает проблемы, с которыми сталкивается преподаватель:

| Боль | Решение |
|------|---------|
| Студенты пишут в разных мессенджерах | Единая точка входа через Telegram |
| Истории сообщений очищаются | Всё сохраняется в БД с таймстемпами |
| Документы разбросаны | Централизованное хранилище MinIO |
| Забываются дедлайны | Автоматические напоминания |
| Повторяющиеся вопросы | FAQ и типовые ответы |
| Сложно отслеживать прогресс | Визуализация milestone'ов |
| Нет времени на первичную проверку | AI-скрининг работ |

## 🤖 Telegram-бот: Команды

### Для студентов:

| Команда | Описание |
|---------|----------|
| `/start` | Первый вход, регистрация |
| `/help` | Справка по командам |
| `/status` | Мои работы и статусы |
| `/deadlines` | Мои дедлайны |
| `/submit` | Сдать работу |
| `/history` | История коммуникаций |

### Для преподавателя:

| Команда | Описание |
|---------|----------|
| `/admin` | Панель управления |
| `/broadcast` | Массовая рассылка |
| `/stats` | Статистика загрузки |
| `/settings` | Настройки AI-скиллов |

## 🧠 AI-скиллы

### 1. Антиплагиат
- Детекция AI-генерированного текста
- Оценка заимствований
- Рекомендации по уникализации

### 2. Структура
- Проверка научной новизны
- Наличие методов исследования
- Научный инструментарий
- Логика изложения

### 3. Оформление
- Соответствие ГОСТ
- Требования к курсовым/ВКР
- Требования журналов (загружаемые)

## 📦 Быстрый старт

```bash
# 1. Клонирование
git clone https://github.com/voodoo2serg/DigitatalTutor.git
cd DigitatalTutor

# 2. Установка (интерактивный мастер)
sudo ./install.sh

# 3. Настройка БД
cd /srv/teaching-system && ./master-config/wizard.sh

# 4. Настройка бота
./bot/admin.py --setup
```
