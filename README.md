# DigitalTutor

**Система управления учебными проектами для преподавателей вузов**

> Персональный помощник для отслеживания работ студентов, курсовых, ВКР, научных статей и проектной деятельности. Telegram-бот + веб-интерфейс + AI-проверка.

---

## Что это такое

DigitalTutor — это комплексное решение для преподавателей, которое объединяет в себе несколько инструментов. Студенты взаимодействуют через Telegram-бот: регистрируются, сдают работы, получают обратную связь. Преподаватель проверяет работы через Telegram или веб-панель. Система автоматически проверяет оригинальность текста, структуру документа и соответствие оформлению с помощью AI.

---

## Возможности

### Для студентов (Telegram-бот + Веб)
- Регистрация и профилирование (ФИО, группа, курс, тип работы)
- Сдача работ с файлами (DOCX, PDF, ODT)
- Отслеживание статуса работ и дедлайнов
- Переписка с преподавателем по каждой работе
- Веб-доступ: архив сообщений, файлы, справка (1.5 часа сессия)
- Получение кода через бота или QR-код (для тех, у кого нет Telegram)

### Для преподавателя (Telegram + Веб-панель)
- Управление списком студентов
- Проверка, оценка и комментарий к работам (по 3 шкалам: классическая 1-5, 100-балльная, буквенная A-E)
- Массовая рассылка студентам
- Проводник файлов: дерево папок (Группа / Студент / Тип работы / Файлы)
- AI-анализ работ: оригинальность, структура, оформление (Cerebra / OpenRouter / Ollama / HuggingFace)
- Генерация QR-кодов доступа для студентов без Telegram
- Настройка AI-провайдеров из админ-панели

### Технические особенности
- Docker Compose развёртывание (Debian 13 / VDS)
- PostgreSQL 15 — хранение всех данных
- Redis — кэширование и очереди
- Яндекс.Диск — резервное хранение файлов
- Nginx — обратный прокси
- APScheduler — автоматические напоминания о дедлайнах

## Быстрый старт

```bash
# 1. Клонирование
git clone https://github.com/voodoo2serg/DigitatalTutor.git
cd DigitatalTutor

# 2. Настройка переменных окружения
cp .env.example .env
# Отредактируйте .env: BOT_TOKEN, JWT_SECRET, MASTER_CODE и т.д.

# 3. Запуск через Docker Compose
docker compose up -d

# 4. Веб-интерфейс
# Студенты: http://your-domain:3000
# Админ:    http://your-domain:3000 → "Вход для преподавателя"
# API Docs: http://your-domain:8000/docs
```

## Веб-интерфейс

### Доступ студента
1. Получить одноразовый код через Telegram-бота
2. Ввести код на странице входа (или отсканировать QR-код, выданный преподавателем)
3. Сессия длится 1.5 часа
4. Доступно: работы, переписка, файлы, справка

### Доступ преподавателя
1. Ввести мастер-код (настраивается в .env: `MASTER_CODE`)
2. Сессия длится 8 часов
3. Доступно: студенты, работы, проводник файлов, AI-анализ, QR-коды, настройки

### Безопасность
- AI-проверка результатов **не видна** студентам
- Студент видит только: статус, комментарии преподавателя, дедлайны
- Преподаватель видит: AI-баллы, антиплагиат, полную аналитику
