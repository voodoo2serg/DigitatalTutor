## BUG-001: Кнопки дедлайна не работают
**Дата:** 2026-04-05 22:11
**Статус:** ✅ FIXED
**Серьезность:** CRITICAL

### Описание
При нажатии на кнопки дедлайна ("⚡ Супер срочно", "🎓 Май", "🗓️ Через неделю") бот возвращает ошибку: "❌ Неверный формат даты"

### Шаги воспроизведения
1. Нажать "➕ Сдать работу"
2. Выбрать тип работы
3. Ввести тему
4. Ввести описание
5. Нажать "⚡ Супер срочно"

### Ожидаемый результат
Переход к шагу загрузки файла

### Фактический результат
"❌ Неверный формат даты. Используйте ДД.ММ.ГГГГ"

### Причина
В `submit.py` не было обработчиков для кнопок дедлайна — только для текстового ввода даты.

### Исправление
Добавлены 4 обработчика:
- `deadline_urgent` — для "⚡ Супер срочно"
- `deadline_may` — для "🎓 Май"
- `deadline_week` — для "🗓️ Через неделю"
- `deadline_manual` — для "📅 Указать дату"

### Проверка исправления
- [x] Тест пройден (вручную Владельцем)
- [ ] Автотест создан (в процессе)

## BUG-002: Несоответствие кнопки "Да" и обработчика
**Дата:** 2026-04-05 22:11
**Статус:** ✅ FIXED
**Серьезность:** CRITICAL

### Описание
После заполнения всех полей при нажатии "✅ Да" ничего не происходит.

### Шаги воспроизведения
1. Пройти все шаги сдачи работы
2. Дойти до подтверждения
3. Нажать "✅ Да"

### Ожидаемый результат
Работа сохраняется, появляется сообщение об успехе

### Фактический результат
Ничего не происходит (Update is not handled)

### Причина
Несоответствие текста:
- Клавиатура `get_yes_no_menu()` создает кнопку "✅ Да"
- Обработчик `confirm_submit` ждал "✅ Подтвердить"

### Исправление
Изменен фильтр обработчика:
```python
# Было:
@router.message(SubmitStates.waiting_confirm, F.text == "✅ Подтвердить")

# Стало:
@router.message(SubmitStates.waiting_confirm, F.text == "✅ Да")
```

### Проверка исправления
- [x] Тест пройден (вручную Владельцем)
- [ ] Автотест создан

## BUG-005: Ошибка сохранения работы — неверное поле yandex_file_path
**Дата:** 2026-04-05 23:15
**Статус:** ✅ FIXED
**Серьезность:** CRITICAL
**Найдено:** Агентом во время тестирования (подтверждено скриншотом Владельца)

### Описание
При нажатии "✅ Да" для подтверждения сдачи работы появляется ошибка:
"❌ Ошибка при сохранении работы. Попробуйте позже."

### Лог ошибки
```
Error saving work: 'yandex_file_path' is an invalid keyword argument for StudentWork
```

### Причина
Код пытался установить поле `yandex_file_path` на модели `StudentWork`, 
но это поле есть только в модели `File`.

### Исправление
1. Убрано `yandex_file_path` из создания StudentWork
2. Добавлено создание отдельной записи `File` с правильными полями:
   - `storage_type='local'`
   - `storage_path=file_path`
   - `yandex_file_path=file_path` (для совместимости)

### Проверка исправления
- [ ] Тест пройден
