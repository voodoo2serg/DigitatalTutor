# Legacy Files

These files are from earlier versions of DigitalTutor and are no longer used.
They are kept for reference only.

## Files:
- `welcome.py` - Old standalone registration wizard (python-telegram-bot)
- `admin.py` - Old standalone admin panel (python-telegram-bot)
- `telegram_bot.py` - Original bot prototype (python-telegram-bot)
- `bot_v2_2_hotfix.py` - Hotfix script (python-telegram-bot)
- `bot_v2_backup*.py` - Backup copies of bot entry point
- `bot_v2.py.backup*` - Backup copies

## Note:
These files use `python-telegram-bot` library while the current bot uses `aiogram 3.x`.
Do NOT mix these two libraries in the same runtime.
